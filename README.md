# Yliopistosimulaattori

**Aihe:** Yliopistosimulaattori-peli.

Toteutetaan peli joka simuloi yliopiston haasteita. Pelissä pelaaja on opiskelija, mahdollisesti käpistelijä, jonka tarkoituksena on tasapainotella datauksen, opintojen, sosiaalisen elämän ja unentarpeen välillä.

Peli on RPG-tyylinen ja tekstipohjainen (ASCII-art?). Laajennusmahdollisuuksia peleissä yleensäkin on paljon, sillä uusia skenaarioita voi lisätä mielensä mukaan samoin kuin teknisiä palikoita, kuten online-mahdollisuus (MUD?).

**Käyttäjät:** Pelaaja.

**Kaikkien käyttäjien toiminnot:**

* Pelin aloittaminen.
* Pelin tallennus ja lataus.
* Pelissä liikkuminen.
* Interaktio NPC:n kanssa.
* Interaktio maailmassa olevan objektin kanssa.
* ...
